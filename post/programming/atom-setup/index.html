These are my recommended and day to day packages and themes for a good experience using atom while developing with Javascript. Your IDE will be ready for compiling SASS, language linting, …

I’m looking forward to read your suggestion on the comments

Atom setup for Javascript development

Software required

If you haven’t, grab and install a copy of both atom and nodejs

Global libraries

  
    npm install -g node-sass

Theme:

Seti-UI + Monokai-seti

  
    apm install seti-ui
    apm install monokai-seti
  1. Restart atom (ctrl + shift + f5)
  2. open settings (ctrl/cmd + ,), navigate to themes
  3. Set seti ui for UI Theme
  4. Set monokai-seti ui for Syntax Theme

Custom stylesheet

Atom supports custom styling using css. Find the stylesheet item in the file menu. This is my personal css configuration in case you need some ideas

  
    // style the background color of the tree view
    .tree-view {
      // background-color: whitesmoke;
      font-size: 11px;
    }

    // style the background and foreground colors on the atom-text-editor-element itself
    atom-text-editor {
      // color: white;
      // background-color: hsl(180, 24%, 12%);
      font-size: 13px;color: white;
    }

    // style UI elements inside atom-text-editor
    atom-text-editor .cursor {
      // border-color: red;
    }

    .tree-view .name.icon::before {
      margin: 0 6px 0 0px;
    }

    .list-tree li:not(.list-nested-item):not(.tabBlender),
    .list-group li:not(.list-nested-item):not(.tabBlender),
    .list-tree li.list-nested-item > .list-item,
    .list-group li.list-nested-item > .list-item {
      line-height: 24px;
    }

    .has-custom-icons .tab-bar:not([data-reactid]) .tab .title[data-path]::before {
      margin-right: 5px;
    }
    .tab-bar {
      height: 33px;
      line-height: 30px;
    }
    .tab-bar .tab.active {
      background: brown;
      max-width: 140px;
      min-width: 140px;
    }
    .tab-bar .tab {
      min-width: 4rem;
      max-width: 7rem;
    }

Packages:

Once Atom Package Manager (APM) is installed, every other package can be installed from the command line. Unfortunatelly, APM will install one package at a time (Until this post was published)

Pigments

Pigments provides a preview of html colors, usefull on css files

  
    apm install pigments

Todo show

  
    apm install todo-show

Git time machine

  
    apm install git-time-machine

Emmet

One of the most installed package. It autocompletes your HTML according to a specific format.

  
    tr>td*4>a (shift + space)

Above emmet command will expand into the following HTML markup

  
    
      <tr>
        <td><a href=""></a></td>
        <td><a href=""></a></td>
        <td><a href=""></a></td>
        <td><a href=""></a></td>
      </tr>

Install it using the next command

  
    apm install emmet

Minimap

Installs a map preview of the whole file

  
    apm install minimap

Highlight-selected

Highlights all the ocurrences of the current selected text. You’ll also want to install minimap-highlight-selected which provides support for highlighting on minimap

  
    apm install highlight-selected
    apm install minimap-highlight-selected

Snippets packages

Contribute to the snippets package with some more specific snippets

  
    apm install javascript-snippets
    apm install react-snippets

REPL

Execute Javascript directly in Atom

  
    apm install node-repl
    apm install script

Git plus

Have a better Git support

  
    apm install git-plus

Atom shortcuts

A cheat sheet for atom shortcuts (ctrl + `)

  
    apm install atom-shortcuts

Sync settings

Sync you settings in a gist!

  
    apm install sync-settings

Linters

Linters highlight mistakes directly inside the IDE

  
    npm install -g eslint
    apm install linter
    apm install linter-eslint

React useful packages

  
    apm install language-babel
    apm install react
    apm install autoclose-html

Also have a look to this gist for supporting emmet in jsx

Other useful packages

  
    apm install css-comb
    apm install imdone-atom
    apm install tasks
    apm install docblockr
    apm install linter-flow
    apm install open-recent
    apm install tool-bar
    apm install atom-beautify
    apm install atom-ternjs
    apm install autocomplete-modules
    apm install emmet-snippets-compatibility

Other suggested packages

If you program with react native or ….

Nuclide

Nuclide provides debug, worksets, …

  
    apm install nuclide
  • open settings (ctrl/cmd + ,) > packages > look for nuclide > settings > Install recommended packages on start
  • Restart Atom
